# Resnet on Cifar10

## Requirements

* Python 3.5
* tf-1.9


## Getting started
### Preparing data
Download MINST dataset and extract to directory ``` ./data/mnist```.
```shell
./MNIST_data
    t10k-images-idx3-ubyte.gz
    t10k-labels-idx1-ubyte.gz
    train-images-idx3-ubyte.gz
    train-labels-idx1-ubyte.gz
```

### Run and Plot
```shell
python mnist_logistic.py  
```
